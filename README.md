# boostPythonExample
Examples to use boost.python for extending c++ to python

Environment:
    Ubuntu Linux 15.10 (x86_64)

Tools:
    CMake3.2 
    gcc5.2
    python2.7, 3.4, 3.5
    boost1.58

Notes:
    The default bjam for boost.python is not used here; CMake and gcc are combined to manage and build the project.
